# Data

All information utilized within this study is owned by Signify EMEA and protected by the applicable confidentiality agreements. No raw or processed datasets are included in this repository.

## Data Source
- **System:** SAP PS Module
- **Report:** Monthly Deepdive Export (.xlsm)
- **Storage:** Microsoft SharePoint (Signify EMEA — internal access only)

## Dataset Structure
The dataset that is created within an analytical format is built with a Longitude Create Panel of weekly Deepdive snapshots at the WBS level, and each row represents a singular identifiable WBS element on a specific reporting deadline.

### Variable Categories
- **Temporal:** Planned Recognition Date, Actual Recognition Date
- **Identification:** Project Definition, WBS Element, Sales Document, Order, 12NC Code
- **Operational:** WBS Status, Delivered Quantity, Open POs, Total Hours, PM Hours, SE Hours
- **Financial:** Actual Cost, Cost Budget, IGM%, Material Cost, Labour Cost, External Cost, EaC Costs

## Access
Data is not publicly available. For academic review purposes, please contact the author directly.
